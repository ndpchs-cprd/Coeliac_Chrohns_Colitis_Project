# Coeliac_Chrohns_Colitis_Project
Codelists for conditions of interest in a Coeliac UK and Crohn’s &amp; Colitis UK potential project.

Each Codelist is available in EMIS, SNOMED, ICD10 and ICD11 coding systems.

Codes have been appended with "a" and "b" to avoid accidental truncation when opened in Microsoft Excel.
